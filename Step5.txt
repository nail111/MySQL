Выбор уникальных полей:
--------------------------------------------------------------------------------------------------

SELECT DISTINCT [name_of_variable] FROM [name_of_table];

--------------------------------------------------------------------------------------------------

Выбор определенное количество полей:
--------------------------------------------------------------------------------------------------

SELECT * FROM [name_of_table] LIMIT [number];
Example:
SELECT * FROM users LIMIT 2;

--------------------------------------------------------------------------------------------------

Сортировка:
--------------------------------------------------------------------------------------------------

SELECT * FROM users WHERE id>2 ORDER BY id LIMIT 3;

SELECT * FROM users WHERE id>2 ORDER BY id DESC LIMIT 3;

--------------------------------------------------------------------------------------------------
